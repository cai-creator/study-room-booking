## 二、JDK 版本兼容问题修复

### 问题现象
运行主类报错：
```
java.lang.ExceptionInInitializerError
com.sun.tools.javac.code.TypeTag :: UNKNOWN
```

### 原因
系统默认 JDK 为 25，Lombok 1.18.30（Spring Boot 3.2.5 默认版本）不支持 JDK 25。

### 解决方案
1. **下载并安装 JDK 21** 到 `E:\Java\jdk-21.0.5+11`（Eclipse Temurin 21.0.5）
2. **升级 Lombok 版本**：`pom.xml` 中显式指定 `lombok.version=1.18.36`
3. **Java 版本调整**：`pom.xml` 中 `<java.version>` 从 `17` 改为 `21`
4. **IDEA 配置**：切换项目 SDK 为 JDK 21

## 三、YAML 配置语法修复

### 问题
启动报错：
```
org.yaml.snakeyaml.parser.ParserException: while parsing a block mapping
expected <block end>, but found ','
```

### 原因
`application.yml` 中 `springdoc.group-configs.paths-to-match` 使用了逗号分隔的多值写法，YAML 不支持。

### 修复
将逗号分隔改为标准 YAML 数组格式：
```yaml
paths-to-match:
  - '/auth/**'
  - '/users/**'
```

## 四、端口冲突解决

### 问题
启动报错：`Port 8080 was already in use`

### 原因
端口 8080 被 NI WebServer（Multisim 软件）占用。

### 解决方案
将 `application.yml` 中 `server.port` 从 `8080` 改为 `8081`。

## 五、项目配置切换
